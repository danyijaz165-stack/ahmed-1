# Quick Setup Guide

## Installation Steps

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open Browser**
   Navigate to: http://localhost:3000

## Available Pages

- **Homepage**: `/`
- **Catalog**: `/collections/catalog`
- **Gentleman's Reserve**: `/collections/gentlemans-reserve`
- **Products**: `/collections/products`
- **Contact**: `/pages/contact`
- **Cart**: `/cart`
- **Checkout**: `/checkout`
- **Login**: `/account/login`
- **Product Pages**: `/products/[slug]`
